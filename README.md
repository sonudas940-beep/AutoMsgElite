# AutoMsg Elite CRM

Local WhatsApp CRM prototype with admin and customer portals.

## Open

Open `index.html` in a browser, or serve this folder with any static server.

Demo admin login:

- Username: `admin`
- Password: `admin123`

Demo customer login:

- Username: `customer`
- Password: `123456`

## Included

- Admin overview dashboard
- WhatsApp QR refresh UI
- Customer create form with login ID, password, plan and expiry date
- 7 days trial, 6 months and custom expiry support
- Per-customer feature permissions
- Expired customer lock screen
- Customer portal with AutoMsg Elite menu
- Connect Account for WhatsApp, Instagram, Facebook and Telegram
- Message bot, template bot, templates, campaigns, bulk campaigns, activity log, chat, canned reply, AI prompts and settings screens

Data is saved in browser localStorage. A production version needs a real backend database and WhatsApp/Meta/Telegram API integration.
